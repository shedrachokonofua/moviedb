# moviedb

## Requirements
Node 8+

## Installation
Install dependencies
```bash
$ npm install
```

Create env file
```bash
$ cp .env.example .env
```

Set API KEY in .env file
```
API_KEY=[INSERT_YOUR_KEY_HERE]
```

## Usage
Start Server
```bash
$ npm start
```

## Testing
Generate UI Component Storybook
```bash
$ npm run storybook
```

Run Unit Tests
```bash
$ npm test
```
